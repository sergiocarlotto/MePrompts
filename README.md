## Como usar

### Claude Code

No `CLAUDE.md` do projeto, adicione referências com `@`:

```markdown
@C:\Users\sergi\OneDrive\Documents\GitHub\MePrompts\projects\pacs\context.md
@C:\Users\sergi\OneDrive\Documents\GitHub\MePrompts\projects\pacs\interactions.md
```

### Outras ferramentas (Cursor, ChatGPT, etc.)

Copie o conteúdo do arquivo desejado e cole no início da conversa ou campo de system prompt da ferramenta.
