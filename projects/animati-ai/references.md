# Materiais de Referência — animati-ai

## Checklist de Configuração de IA para Novos Clientes

Referência: [ANAP-1300](https://animati.atlassian.net/browse/ANAP-1300)

### 1. Habilitar o Agent (transcrição de laudos narrados)

1. Acessar o menu `Administrar → Usuários`
2. Localizar o usuário e entrar no cadastro
3. Clicar em `Preferências da IA`
4. Habilitar o Agent e clicar em `Salvar`

> Após habilitado, os próximos laudos narrados (áudios) enviados pelo usuário serão transcritos pelo Agent.

### 2. Habilitar o Copilot (Assistente de Laudos)

1. Garantir que o cliente está utilizando a nova central de laudos
2. Acessar o item `Assistente de laudos` na central

### Observações

- A configuração do Agent é feita por usuário, não globalmente
- O serviço `animati-ai` deve estar habilitado na infraestrutura do cliente antes de iniciar a configuração
